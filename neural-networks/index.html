Neural Networks

Consistent with recent deep learning trends, the Rust community has followed suit by building several experimental neural networks.

Leaf made a huge splash in the community and included great resources like the Leaf Book and talks like this one. But the team behind Leaf moved on, and despite over 4k GitHub stars has yet to find a new maintainer.

Taking a page from pythonâ€™s skikit-learn project, rusty-machine is one of the most actively developed ML crates in the Rust ecosystem. It also powers Learning Machines, a set of interactive tutorials on machine learning, and this presentation provides a solid introduction.

Other neural network libraries have been created, but seem to mostly exist as experimental projects. Some of the projects listed hear appear to be inactive, but are still listed to serve as an inspiration for any future projects.

leaf

Machine Learning Framework for Hackers

rusty-machine

A machine learning library.

rustml

A library for doing maching learning in Rust.

nn

A multilayer feedforward backpropagation neural network library.

tensorflow

Rust language bindings for TensorFlow. This project is still under active development and not yet ready for widespread use.

hal-ml

A machine learning library that is able to run on Nvidia, OpenCL or CPU BLAS based compute backends.

fann

Wrapper for the Fast Artificial Neural Networks library

deeplearn

Deep neural networks in Rust
