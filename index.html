Are we learning yet?

The comprehensive guide to
          the state of machine learning in Rust

You can accomplish a lot,
but the ecosystem isn't mature yet.

Rust's performance, low-level control, and zero-cost high-level abstractions make it a compelling alternative to more established ecosystems for Machine Learning. The Rust ML ecosystem is still young, however several ambitious projects and a variety of building blocks have emerged and using Rust to solve a real-world machine learning problem even made the cut for RustConf 2016.

Still, much of the Rust ML ecoystem is best described as experimental. So there are plenty of opportunities for new projects to fill in the gaps.

For help or questions with Rust ML, reach out on #rust-machine-learning.
